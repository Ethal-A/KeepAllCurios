# KeepAllCurios
A lightweight server-side Minecraft forge mod that allows players to keep all their curios from the [Curios API mod](https://www.curseforge.com/minecraft/mc-mods/curios).

## Maintenance
This mod will not be maintained. You can test it for different Minecraft versions but it was designed for 1.20.1.
Feel free to also fork this mod and make your own version of it.
